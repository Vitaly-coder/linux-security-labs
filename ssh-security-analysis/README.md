# Анализ журнала безопасности SSH

## Обзор
В этом проекте демонстрируется практический анализ событий безопасности, связанных с SSH
, с использованием системных журналов Linux и журнала systemd.

В проекте моделируются неудачные попытки аутентификации по SSH и анализируется, как
регистрируются действия грубой силы на системном уровне.

---

## Статус службы SSH

Служба SSH изначально была отключена и неактивна.

После активации вручную служба была запущена и включена с помощью systemd,
что увеличило вероятность атаки на систему.
