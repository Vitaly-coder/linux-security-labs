# Анализ системных журналов Linux

## Обзор
Этот проект демонстрирует практический анализ системных журналов Linux, используемых для
мониторинга безопасности и расследования событий, связанных с аутентификацией.

Все действия, представленные в этом проекте, были выполнены вручную в системе Linux
и подтверждены с помощью системных журналов.

---

## Обзор источников журналов

### /var/log/syslog
Содержит общие события системного уровня, служебные сообщения и операционные действия.

### /var/log/auth.log
Содержит критически важные для безопасности события, связанные с:
- аутентификацией пользователя
- использованием sudo
- повышением привилегий
- сбоями аутентификации

### журнал systemd (journalctl)
Централизованная система ведения журнала, используемая systemd для хранения событий от служб,
ядра и подсистем аутентификации.
