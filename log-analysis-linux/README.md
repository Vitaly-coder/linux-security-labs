# Анализ системных журналов Linux

## Цель проекта
Этот проект демонстрирует практическую работу с системными журналами Linux
, используемыми для мониторинга безопасности и анализа инцидентов.

## Область применения
- системные журналы (`syslog`, `сообщения`)
- журналы аутентификации ("auth.log`)
- журнал systemd (`journalctl`)

## Инструменты
- Linux (на базе Debian)
- journalctl
- стандартные утилиты ведения журнала (tail, grep)
